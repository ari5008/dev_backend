# Name）
 ~ 曲選曲 ~
 
曲を選択し、閲覧できる掲示板
 
# DEMO
 
いろんな曲を知るきっかけになる。日々の生活に癒しを求めている人におすすめ
 
# Features
 
Login機能、アカウント編集機能、曲の並び替え、いいね機能など
